# 🚀 About Me

PHP 백엔드 필드에서 10년 넘게 구르며 서비스의 시작과 성장을 함께해 왔습니다. Laravel이나 CodeIgniter 같은 프레임워크를 단순히 사용하는 수준을 넘어, 쇼핑몰 빌더부터 CMS, 템플릿 엔진까지 비즈니스의 뼈대가 되는 핵심 로직들을 직접 설계하고 구현해 본 경험이 제 큰 자산입니다.

리눅스 서버 환경이나 데이터베이스 운영은 이미 몸에 익은 일상이고, GitHub나 Jira를 통한 팀 협업 역시 익숙합니다. 하지만 과거의 익숙함에만 머물지는 않습니다. 최근에는 Python(FastAPI)과 NestJS(TypeScript)를 깊게 파고 있는데, 현대적인 웹 아키텍처와 더 정교한 API 설계법을 익히는 과정에서 개발자로서의 시야가 다시 한번 넓어지는 중입니다.

저는 '그저 돌아가는 코드'를 짜는 데 만족하지 않습니다. 유지보수가 쉽고 성능이 확실한 시스템, 그리고 팀과 비즈니스에 실질적인 가치를 더하는 결과물을 만드는 데 집중합니다.

> **Role:** Backend Lead Developer (기여도: 00%)
> **Description:** 누구나 쉽게 쇼핑몰을 구축할 수 있는 빌더 서비스를 개발했습니다.
- **Problem:** 기존 빌더들의 복잡한 UI와 느린 템플릿 로딩 속도 문제 해결 필요.
- **Solution:** 독자적인 템플릿 엔진을 설계하여 렌더링 속도를 30% 개선하고, 직관적인 드래그 앤 드롭 인터페이스를 위한 API를 구축.
- **Tech Stack:** PHP, CodeIgniter, MariaDB, jQuery
