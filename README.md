# Backend Developer — Commerce Platform · Builder Systems · Operational Automation

안녕하세요. 저는 **커머스 플랫폼과 빌더형 솔루션 환경**에서 백엔드 기능 개발, 운영 안정화, 외부 연동 구조 설계를 수행해 온 개발자입니다.
복잡한 비즈니스 규칙과 레거시 제약을 그대로 두지 않고, **운영 가능한 구조와 유지보수 가능한 로직**으로 정리하는 일을 주로 해왔습니다.
최근에는 **TypeScript/NestJS**, **Python/FastAPI** 기반 API 설계와 자동화, 검색·추천·AI 보조 기능 확장에도 관심을 두고 있습니다.

> 기능 추가보다, 서비스 운영에 실제로 남는 구조를 만드는 일을 중요하게 생각합니다.

---

## What I Work On

- **Commerce Backend**: 주문, 배송, 정산, 상품 운영처럼 비즈니스 규칙이 많은 영역의 서버 로직 설계 및 개선
- **Builder & Legacy Modernization**: 빌더/CMS/게시판 기반 서비스 확장, 레거시 구조 개선, 운영 가능한 흐름 재설계
- **Integration & Automation**: 외부 API 연동, 운영 도구 개발, 반복 업무 자동화 구조 설계
- **Operational Stability**: 장애 대응, 버그 수정, 호환성 개선, 운영 효율을 높이는 구조 정리

### Commerce Platform
커머스 서비스 환경에서 주문, 배송, 정산, 상품 운영과 관련된 백엔드 기능을 설계·개선해 왔습니다.
단순 기능 추가보다 정책, 예외 처리, 운영 흐름을 서비스 로직과 데이터 구조로 정리하는 작업을 주로 수행했습니다.

### Builder & Legacy Systems
빌더형 솔루션과 레거시 구조 위에서 신규 기능을 추가하고, 기존 운영 방식을 크게 해치지 않으면서 확장성과 유지보수성을 높이는 작업을 경험했습니다.
게시판, 스킨, 플러그인 구조를 활용한 업무 시스템 확장과 운영 이슈 대응, 마이그레이션 처리도 함께 수행했습니다.

### Integration, Automation, and AI-Assisted Features
외부 서비스 연동, 운영 자동화, API 중심 구조 설계와 함께 검색·추천·AI 보조 기능 고도화에도 관심을 확장하고 있습니다.
반복 업무를 줄이고 운영자가 다루기 쉬운 흐름을 만드는 방향으로 내부 도구와 자동화 구조를 설계해 왔습니다.
